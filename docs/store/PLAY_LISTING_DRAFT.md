### 短い説明(80文字以内)

### 詳しい説明(4000文字以内、ドラフト)
```
乗った路線が光る、自分だけの日本地図を育てよう。

「路線埋め立てマップ」は、日本全国597路線・27,725kmに対応した鉄道乗りつぶしアプリです。
地図上の路線をタップして「乗った！」を押すだけで、その路線が光って塗りつぶされます。

【特徴】
✓ 全597路線対応(国土数値情報データ使用)
✓ 路線の一部だけ乗った場合も区間単位で記録できる
✓ 都道府県別・路線種別(新幹線/JR/私鉄/地下鉄)の達成率を統計画面で確認
✓ 称号(アチーブメント)を集める楽しみ
✓ 達成状況をシェア画像として保存・SNS投稿
✓ データはすべて端末内に保存。サーバーへの送信なし、アカウント登録不要
✓ オフラインでも利用可能

鉄道好き・乗りつぶし好きの方はもちろん、通勤通学路線から始めて
少しずつ地図を塗り広げていく達成感を楽しめます。
```
